## 🎬 二、客戶會體驗到什麼(必懂)

服務人員要先理解客戶看到什麼,才知道自己要做什麼。

```
[客戶坐到 A05 桌]
    ↓
[掃桌邊立牌 LINE QR]
    ↓
[加 IC 高雄 LINE 官方帳號] ← bot 回「您已綁定 A05 桌」
    ↓
[用餐]
    ↓
[結帳]
    ↓
分兩條路:
   ┌── A. 紙本發票客戶(自助,你不用做事)
   │      → LIFF 內按「掃發票拿擲骰」→ 進遊戲擲骰
   │
   └── B. 統編/載具客戶(沒紙本,需要你幫忙)
          → 跟你說「我要玩遊戲」
          → 你開 /admin/tables → 點 A05 → 輸金額 → 確認 ✅
          → 客戶 LINE 收到通知 → 點訊息進遊戲擲骰
    ↓
[擲骰 → 走棋盤 → 領獎勵券 → 到第 15 格兌換大獎]
```

**金額換骰子規則:每 NT$2,000 = 1 次骰子,上限 5 次**
- $1,999 = 0 次(不能玩)
- $2,500 = 1 次
- $5,000 = 2 次
- $10,000 = 5 次(達上限)


---

### 每次結帳要做的事(只在客戶說「我要玩遊戲」且沒紙本發票時)

#### 客戶有紙本發票 → **你不用做任何事**
客戶自己掃發票就會擲骰,你只要確認他開心離開就好。

#### 客戶用統編 / 載具(沒紙本) → 你要幫一下

1. **開 admin LIFF**
   - LINE → IC 高雄官方帳號 → Rich Menu / 連結 → `/admin/tables`
   - 或直接打開 bookmark

2. **選餐廳**
   - 頁面頂部 5 個餐廳 tab(ZL/WR/SD/HW/BL)
   - 點你工作的那間

3. **找客戶的桌號**
   - 桌號 grid 上顯示:
     - ⚪ 灰色 = 等待中(客戶還沒加 LINE / 還沒掃桌邊 QR)
     - 🟢 綠色 = **客戶已綁定**(你可以點來啟用)
     - 🟡 黃色 = 已啟用(剛剛按過,客戶 LINE 已收到通知)
     - ⚫ 灰底 = 冷卻中

4. **點 🟢 那一桌**
   - 跳出 modal,顯示綁定客戶的 LINE 頭像 + 名字 → **目視確認是不是這位客戶**
   - 在金額欄位輸入帳單總金額(例如 `2400`)
   - 系統會顯示「將給予客戶 N 次擲骰機會」(自動算)
   - 點 **「✓ 啟用並推送」**

5. **完成 ✅**
   - 客戶 LINE 立刻收到一張 Flex Message
   - 他點「立即遊玩」就進遊戲
   - 桌號變成 🟡 已啟用

#### ⚠️ 如果客戶說「我已加 LINE 但桌號是 ⚪ 找不到我」
代表客戶**沒掃桌邊立牌 QR**(只是普通加好友)。
- 請他**對著桌邊立牌再掃一次 QR**(必須帶桌號 deep link)
- 他會收到 LINE 訊息「您已綁定 A05 桌」就 OK 了
- 你再去 admin 點 🟢 啟用

### 異常處理

| 狀況 | 處理 |
|---|---|
| 客戶掃發票顯示「已被使用」 | 該發票確實被人 redeem 過(可能客戶之前掃過、或同事誤掃)。**請改走 admin 啟用流程**。一張發票終身只能用一次。 |
| 客戶掃發票顯示「需先綁桌」 | 客戶超過 60 分鐘沒掃桌邊 QR。請他**現在掃桌邊立牌**重新綁定,然後再掃發票。 |
| 客戶掃發票顯示「金額不足」 | 金額 < NT$2,000。確認帳單金額是否正確,若客戶確實未滿 2000 則不能玩。 |
| 客戶說「我達 15 格了想再玩」 | 下次他來消費,你 admin 啟用時系統會**自動歸零保留歷史**,他從 0 重新走,但 earned_rewards 不會清。 |
| 客戶手機沒 LINE | 引導下載 LINE。若客戶堅持不裝,這活動無法參與 — 委婉婉拒。 |
| admin LIFF 進不去顯示「請聯絡 IT 加入」| 表示你還沒被加到 staff_whitelist。找 Tony 或 IT。 |

## 📊 四、行銷部 SOP

### 每天看的事(2 分鐘)

進 `/admin/customers`,看頂部 6 個 KPI 卡片:
- 總客戶 / 7 日活躍 / 30 日活躍 / 累計消費 / 總擲骰 / 總領獎

關注什麼:
- **7 日活躍** 是否上升 → 活動有沒有 traction
- **30 日活躍 vs 總客戶比率** → 留存率
- **By Restaurant 業績排行** → 哪間餐廳最熱 / 最冷

### 每週看的事(15 分鐘)

1. 排序「**累計消費 DESC**」→ 看 VIP top 50
2. 排序「**首次互動 ASC**」→ 看老客戶有沒有持續來
3. 點個別客戶 → 看 timeline(誰來哪餐廳幾次、領過什麼獎)

### 每月做的事(EDM / 簡訊)

1. 點右上「📥 CSV 匯出」→ 下載 `ic-kaohsiung-customers-YYYY-MM-DD.csv`
2. 用 Excel 開,UTF-8 中文不亂碼
3. 篩選你要的群(例如 total_visits >= 3 = 回頭客)
4. user_id 給 LINE 官方帳號 Messaging API 做客製化推送
5. (未來功能)直接在 dashboard 點「群發訊息」

### CSV 欄位

| 欄位 | 說明 |
|---|---|
| LINE user_id | 用於 LINE Messaging API push |
| 顯示名稱 | LINE 名字 |
| 首次互動 / 最近活動 | 台北時間 |
| 來訪次數 | activate 計數 |
| 累計消費 | NT$ |
| 擲骰總次 / 領獎總次 / 完成季數 | 互動深度 |

## 🔍 五、客戶常見問題 FAQ(印出來貼櫃台)

**Q1. 怎麼玩?**
A:加我們 LINE 官方帳號(掃桌邊 QR),用餐結帳後即可開始,消費滿 NT$2,000 一次擲骰機會,擲到 15 格可換招牌主餐。

**Q2. 我已經加 LINE 了,還要再掃嗎?**
A:**要,請對著桌邊立牌的 QR 再掃一次**,這樣系統才知道您坐在哪桌。

**Q3. 我用統編開發票,沒有紙本怎麼辦?**
A:結帳時跟我們講一聲,我們幫您啟用。

**Q4. 我可以幫朋友玩嗎?**
A:不行,LINE 帳號跟發票都綁定唯一,一張發票只能一個人用。

**Q5. 我可以下次再玩嗎?**
A:可以,進度會保留。但每次消費滿 NT$2,000 才能繼續擲骰。

**Q6. 達 15 格後可以再玩嗎?**
A:可以,下次消費後系統會給您新的擲骰機會,但兌換的招牌主餐券會保留在您的紀錄裡。

**Q7. 為什麼我擲了 1 點?**
A:擲骰是隨機 1~3 點。

**Q8. 我家人也想玩,可以共用發票嗎?**
A:不行,一張發票一次有效。建議分單結帳。
