**Subject:** Culinary Journey v2 — Room Charge feature + Dashboard upgrades

Dear team,

The Culinary Journey game system just got a meaningful upgrade. Here are the four changes — **please read fully if you're operating staff**; marketing colleagues only need points 3 & 4.

### ✨ Four New Features

1. **🏨 Room Charge Mode (NEW)**
   For hotel guests who sign meals to their room (no 統一發票 issued by the restaurant). Open `/admin/checkout` and **tap the new toggle at the top** to switch from 💳 Regular Checkout to 🏨 Room Charge. No invoice number required; the UI turns indigo so you can't confuse the two modes. Guest still scans the QR to claim dice.

2. **⏱ Clearer QR Expiry Warning**
   Generated QR codes now show an amber banner below counting down the remaining time. **The banner turns red when expired** — staff will never accidentally show a dead QR.

3. **📊 Dashboard Upgrades**
   `/admin/customers` got three new cards:
   - **Conversion funnel** (Add Friend → Bind Restaurant → Redeem → Win Prize)
   - **Per-restaurant activity** (click any row to drill down to that outlet's customers)
   - **Event timeline now differentiates** 🏨 Room Charge vs 💳 Regular Checkout vs 📄 Self-Scan Invoice

4. **🔑 URL Token Sharing** (for managers / marketing partners)
   You can now view the dashboard via a shared URL without needing LINE whitelist. I'll DM the URL individually — **please don't forward it to group chats; the URL is effectively a master key**.
