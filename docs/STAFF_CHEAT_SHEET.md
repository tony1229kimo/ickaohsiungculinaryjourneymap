## ⚡ 您 90% 時間什麼都不用做

客人會自己:加 LINE 好友、綁桌、掃發票或拍小白單、玩遊戲。

---

## 🎯 您只需要在這 3 種狀況介入

### 1️⃣ 客人問「這遊戲怎麼玩?」

> 「請用 LINE 掃這個桌邊 QR 加我們好友,
> 然後對話框打 `table:` 加桌號(例如 `table:ZL05`),
> 系統會傳遊戲連結給您。消費滿 NT$2,000 結帳後就能玩。」

### 2️⃣ 客人卡關 → 走「結帳 QR」6 步

> 客人看到黃色 banner、用載具、發票過期、卡關 → 您主動補發 QR。

```
1. 開連結   .../admin/checkout
2. 輸 PIN   91097496
3. 輸發票編號(小白單上「發票號碼」例:BM36258896)
4. 輸金額  (例:5000)
5. 出 QR    倒數 2 分鐘
6. 客人 LINE 相機掃 → 完成
```

**規則:**
- NT$2,000 = 1 次擲骰,上限 5 次
- 必填發票編號(防雙領)
- QR 2 分鐘失效

### 3️⃣ 客人吵「我這張發票明明沒用過!」→ 客訴查詢

```
1. 開連結   .../admin/lookup
2. 輸 PIN   91097496(同上)
3. 輸發票編號 → 查詢
```

**結果三色:**

- 🟢 **綠 = 未使用** → 客人沒說謊,引導他用結帳 QR
- 🔴 **紅 = 已被使用** → 顯示時間 / 使用者 / 桌號 → **畫面給客人看**
- 🟡 **黃 = 已開 QR 但沒掃** → 請客人立刻掃或等 2 分鐘

## 📋 客人手機錯誤訊息對照

| 看到什麼 | 您怎麼處理 |
|---------|-----------|
| 請先掃描桌邊立牌 | 帶他綁桌 `table:XX` |
| 不是電子發票格式 | 對準發票**左側** QR(不是右側) |
| 不是 IC 高雄洲際開的 | 確認手上發票來源 |
| 發票日期過期 | 走 **結帳 QR** 補發 |
| 已被使用過 | 走 **客訴查詢** 確認 |
| 您這一桌已兌換過一張 | 同桌限一張,真有疑問走客訴查詢 |
| 金額需滿 NT$2,000 | 沒到門檻,告知無法參加 |
| 圖片不夠清楚 / 不像 IC 單據 | 重拍或走 **結帳 QR** |
| 黃色「請洽服務人員」 | 直接走 **結帳 QR** |

→ **幾乎所有狀況最終答案 = 走結帳 QR。**

---

## ❌ 絕對不要

- 不要把 PIN 寫在牆上(口頭傳達就好)
- 不要把 QR 截圖傳給多人(第一個掃的拿到)
- 不要為了討好客人破壞規則
- 不要替客人輸 LINE 帳號 / 密碼
- 不要在沒查證的情況下相信「我朋友剛幫我領了」
